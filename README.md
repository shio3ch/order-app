# ORDER APP -Pelayan-

## Stacks

### Frontend

* Next.js

### Backend

* Nest.js

### DB

* Postgres

## Installation

```sh
pnpm install
```

## Scripts

### Run for Frontend

```sh
pnpm run front start
```

### Run for Backend

```sh
pnpm run back start
```

### Run for Postgres

```sh
docker-compose up -d
```
